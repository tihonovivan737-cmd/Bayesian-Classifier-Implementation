# Зависимости проекта

## Внешние зависимости

У проекта нет сторонних библиотек или пакетов.

## Необходимое окружение

- Операционная система: Windows, Linux или macOS
- Компилятор C++: `g++`, `clang++` или Microsoft Visual C++
- Поддержка стандарта: рекомендуется `C++17` или новее

## Используемые стандартные заголовки

- `<iostream>` — ввод и вывод в консоль
- `<fstream>` — чтение данных из файлов
- `<cmath>` — математические функции (`sqrt`, `pow`, `exp`)

## Файловые зависимости

Для корректной работы рядом с исполняемым файлом должны находиться:

- `iris34_train.txt`
- `iris34_test.txt`

## Команда сборки

### GCC / Clang

```bash
g++ -std=c++17 -O2 -o bayes_classifier main.cpp
```

### MSVC

```powershell
cl /EHsc /std:c++17 main.cpp
```
