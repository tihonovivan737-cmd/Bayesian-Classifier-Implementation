# Байесовский классификатор на C++

Этот проект реализует простой гауссовский байесовский классификатор для набора данных Iris. Программа обучается по файлу `iris34_train.txt`, вычисляет математическое ожидание и дисперсию для каждого класса и признака, а затем выполняет классификацию объектов из выбранного файла.

## Что делает программа

- считывает обучающую выборку из `iris34_train.txt`;
- вычисляет параметры распределения для каждого класса;
- по выбору пользователя обрабатывает `train` или `test` набор;
- выводит предсказанный и фактический класс для каждой строки;
- считает итоговую точность классификации.

## Структура проекта

- `main.cpp` — исходный код программы;
- `iris34_train.txt` — обучающая выборка;
- `iris34_test.txt` — тестовая выборка;
- `DEPENDENCIES.md` — список зависимостей и требований для сборки.

## Требования

Проект не использует сторонние библиотеки. Для сборки нужен только C++-компилятор с поддержкой стандартной библиотеки:

- `g++` / `clang++` / `MSVC`;
- стандартные заголовки C++: `iostream`, `fstream`, `cmath`.

Подробности вынесены в файл [DEPENDENCIES.md](/C:/Users/tihon/PycharmProjects/Bayesian-Classifier-Implementation/DEPENDENCIES.md).

## Сборка

### GCC или Clang

```bash
g++ -std=c++17 -O2 -o bayes_classifier main.cpp
```

### MSVC

```powershell
cl /EHsc /std:c++17 main.cpp
```

## Запуск

После запуска программа попросит указать, какой файл обработать:

- `train` — проверить работу на обучающей выборке;
- `test` — проверить работу на тестовой выборке.

Пример:

```text
Выберите файл для обработки (train/test): test
```

## Формат данных

Каждая строка в файлах данных содержит:

`<признак_1> <признак_2> <класс>`

Пример:

```text
4.7 1.4 1
```

## Ограничения текущей реализации

- используется только 2 признака;
- обучение всегда выполняется по `iris34_train.txt`;
- классы ожидаются в диапазоне `0..2`;
- проект реализован как консольная программа без аргументов командной строки.
